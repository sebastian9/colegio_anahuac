Programas para la elaboración de Páginas Web

Sublime Text Ruby on Rails Dreamweaver Wix Kompozer
Ventajas
  • Es un procesador de texto poderoso y optimizado para la programación
  • Está hecho en python y es extremadamente ligero, en mi opinión tanto como wordpad
  • Resaltado del Código
  • Tiene un teclado predictivo para las etiquetas, los atributos, y las propiedades
  • Te permite editar varias secciones del código simultáneamente (varios cursores)
  • Cuando se usa en conjunto con Firefox Developer Edition o Chrome, la función de "inspect element" te permite entender mucho más fácilmente las repercusiones de tu código en el "layout" y al cambiar ciertas secciones del código o activar y desactivar comandos puedes depurar tu código muy efectivamente.
  • Es un marco de trabajo
  • Funciona con Ruby
  • Puedes integrar bases de datos
  • Puedes crear contenido dinámico
  • Te permite implementar un sistema de manejo de contenidos
  • Se puede integrar con preprocesadores de código como Coffescript, Haml, y Sass
  • Te permite crear aplicaciones web fácilmente escalables en poco tiempo
  • Puedes crear blogs, e-commerce, redes sociales, wikis y diferentes tipos de aplicaciones web.
  • Se le puede trabajar usando Sublime-Text 3
  • No se necesita gran cantidad de conocimientos en programación
  • Puedes visualizar tu trabajo dentro del mismo programa
  • Tienes todo el respaldo de Adobe
  • Es el programa WYSWYG que cuenta con mayor cantidad de herramientas y soporte
  • Te permite insertar imágenes desde el mismo programa
  • Cuenta con una biblioteca de fondos
  • Es posible validar el código en el mismo programa
  • Administrador de archivos, te permite ver el directorio de tu página web y darle un seguimiento a los links en caso de que cambies la localización de un archivo
  • Te permite definir plantillas para implementar un Sistema de Manejo de Contenido
  • Es sencillo, no se necesitan conocimientos técnicos
  • Ofrece hosting inmediato
  • Es online, por lo que siempre tendrás disponibles los archivos
  • Cuenta con miles de plantillas de alta calidad en diseño gráfico
  • Cuenta con soporte si estás en un plan de pagos
  • Puedes crear una página de comercio (no es gratis)
  • El desarrollo es muy rápido y de buena calidad visual siempre y cuando tengas un poco de noción sobre el diseño gráfico
  • Es open-source
  • Requiere pocos conocimientos técnicos
  • Tiene visualización en el mismo programa
Desventajas
  • Son absolutamente necesarios los conocimientos en programación
  • Para visualizar tu trabajo es necesario guardarlo y abrirlo en un navegador
  • Por sí solo sirve únicamente para la creación de páginas estáticas con HTML, CSS, y Javascript.
  • La instalación es complicada, ya que tiene muchas dependencias
  • No funciona con Windows como es esperado
  • Al ser un marco de trabajo, para usarlo debes trabajar con ruby, la terminal, un bloc de notas, un navegador web, un sistema de control de versiones, una base de datos, el explorador de archivos, etc.
  • El nivel de conocimientos necesarios para un manejo adecuado y un aprovechamiento óptimo es el equivalente a casi 1 año de estudio dedicando 10 horas a la semana
  • Presenta resaltado del código
  • Es muy poco accesible
  • Es un programa que demanda muchos recursos de hardware
  • Genera mucho código basura
  • Es una interfaz confusa y un programa que tiene muchas funciones ocultas y desconocidas
  • La visualización inmediata de la página dentro de dreamweaver muchas veces no será lo mismo que interpretará un navegador
  • Los cógdigos de javascript son poco específicos y muy largos
  • Aproximadamente sólo el 10% de las funcionalidades son útiles
  • Te hace flojo e ignorante porque muchas veces no sabes lo que haces o cómo hacer algo en específico.
  • Tu trabajo no es verdaderamente profesional si no conoces sobre programación, un Web Master debe aproximarse a DreamWeaver una vez que entiende completamente cómo es que funciona.
  • Puedes integrar fácilmente redes sociales y google analytics
  • No es profesional
  • Estás sujeto al hosting de wix
  • No tienes control sobre el código generado
  • Tiene muy poca funcionalidad
  • Las páginas creadas son muy simples
  • No es profesional
  • Tiene poca funcionalidad
  • El código generado no es actual
  • Tiene muy poca libertad de acción
  • Si sabes programación es prácticamente un bloc de notas pobre y con funciones inútiles
